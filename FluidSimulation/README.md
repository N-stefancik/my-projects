Fluid Simulation in python. Uses vispy and matplotlib for visualization. In order to create this project I had to skim through many different physics articles and computer graphics articles to settle on the model I wanted to use. In the end I used the standard smooth particle hydrodynamics model which is a computational method commonly used for simulating fluid. For each particle I calculate the forces from other particles in its region using dictionaries to cut down on computation. In order to further speed up the rate of the simulaiton and make it more realistic I used the JIT (just in time) compiling library from numba. 
In order to use this use right click to move your perspectie and shift+right click to create a force on the particles.
